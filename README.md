多分色々間違っているので、心ある人が突っ込んでくれると嬉しいです。  
当面はCNNにフォーカスします。  

R Markdownで作っています。

数式の説明とか図とかはそのうち追記します。

* 大学1年生くらいの数学は頑張って調べられる
